# Shape Drawing Application (C# WinForms)

## Overview
This project was developed as part of a university academic assignment.
It is a C# Windows Forms application that allows users to draw and
manipulate 2D shapes using object-oriented programming principles.

## Features
- Draw basic 2D shapes on a canvas
- Select and manipulate shapes
- Event-driven mouse and keyboard interactions
- Object-oriented design for shape handling
- Simple and user-friendly WinForms interface

## Technologies Used
- C#
- .NET Framework
- Windows Forms (WinForms)
- Object-Oriented Programming (OOP)

## Project Structure
- `Program.cs` – Application entry point  
- `Form1.cs` – Main form logic  
- `Form1.Designer.cs` – UI layout and controls  
- `Form1.resx` – UI resources  
- `.csproj` / `.sln` – Project and solution files  

## How to Run
1. Clone or download this repository
2. Open the `.sln` file in **Visual Studio**
3. Build and run the project

## Notes
This project was completed as part of university coursework to
demonstrate understanding of event-driven programming, OOP,
and basic graphics handling in C#.
